# Crumi (크루미)

> Your Sales Buddy — 함께 성장하는 영업 파트너

사용자 주도로 진화하는 글로벌 CRM 플랫폼

## Quick Start

1. `docs/PRE_WORK_CHECKLIST.md`를 읽고 사전 준비를 완료하세요
2. Google Antigravity에서 이 프로젝트를 열어주세요
3. CLAUDE.md를 읽으면 프로젝트 전체 맥락을 파악할 수 있습니다

## Tech Stack

Next.js 15 · React Native · Supabase · Tailwind CSS · TypeScript
